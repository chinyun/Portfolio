Tripper, a web application that help balance trip budget!

UI/UX
D3.js
React
Node.js
Redux
Postgresql

This is a web application that starts from scratch. It is designed to help users manage their travel budget before and during the trip. On Dahboard, users can set up multiple travel itineraries and manage them separately.
Detailed functions include supplementary travel days and new expenses, view the expenditure / budget ratio, total and detailed budget setting, and expenditure content modification, etc.

這是一個從零開始發想的網頁應用，目的為幫助使用者在行程前及旅遊期間管理旅遊預算，在 Dahboard 使用者可以建立多個旅遊行程並個別管理，詳細功能包括新增旅遊天數、新增支出、查看支出/預算比例、總預算設定、細項預算設定、修改支出內容等。
